# Oh-My-Posh 설치 및 폰트 적용
1) 시작의 검색 (터미널) 
2) 터미널을 "관리자 권한으로 실행"
3) winget install JanDeDobbeleer.OhMyPosh -s winget
4) 설치가 완료되면 터미널을 완전히 껏다가 다시 "관리자 권한으로 실행"
5) 탭옆에 화살표 클릭 -> 설정 -> 좌측 맨 밑 "json 파일 열기" 
6) profiles의 defaults에 입력
7) "defaults": 
        {
            "font": 
            {
                "face": "JetBrainsMono Nerd Font"
            }
        },
8) 저장

# 테마 적용 (aliens 혹은 powerline)
1) 터미널을 "관리자 권한으로 실행"
2) 터미널에서 명령어: notepad $PROFILE 실행 ※ "지정한 경로를 찾을 수 없습니다." 가 뜨는 경우, 아래의 명령어를 실행하고 위의 명령어를 재실행
                : New-Item -Path $PROFILE -Type File -Force
3) 열린 메모장에 입력
oh-my-posh init pwsh --config "https://raw.githubusercontent.com/JanDeDobbeleer/oh-my-posh/main/themes/aliens.omp.json" | Invoke-Expression
4) 저장 후 메모장 종료
5) 다시 터미널을 "관리자 권한으로 실행"
6) 빨간 글씨로 에러 뜰시 명령어: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned
7) 다시 터미널을 "관리자 권한으로 실행"

# 1. 인코딩 및 코드 페이지 설정 (가장 먼저 실행)
$env:LC_ALL='C.UTF-8'
[System.Console]::OutputEncoding = [System.Text.Encoding]::UTF8
$OutputEncoding = [System.Text.Encoding]::UTF8
[Console]::InputEncoding = [System.Text.Encoding]::UTF8
chcp 65001 > $null

# 2. 별칭(Alias) 설정
Set-Alias grep Select-String

# 3. Oh My Posh 초기화 (Aliens 테마)
oh-my-posh init pwsh --config "https://raw.githubusercontent.com/JanDeDobbeleer/oh-my-posh/main/themes/aliens.omp.json" | Invoke-Expression

# 4. 자동완성 및 제안 기능 최적화
if (Get-Module -ListAvailable PSReadLine) {
    Set-PSReadLineOption -PredictionSource History
    Set-PSReadLineOption -PredictionViewStyle ListView
    Set-PSReadLineOption -EditMode Windows
    Set-PSReadLineKeyHandler -Key Tab -Function MenuComplete
}

# 5. 터미널 아이콘
Import-Module Terminal-Icons
